---
title: 2.2.11 逗号表达式
---

逗号表达式的形式是

- `逗号表达式 , 赋值表达式`
- `赋值表达式`

这里将逗号表达式中，','的左边称为操作数1，右边称为操作数2。

逗号表达式的计算是先计算操作数1，然后计算操作数2，最后返回操作数2的值。例如：
```cpp
a = 42, a // 值为 42
```

逗号表达式也可以连续使用，例如：
```cpp
a = 42, b = 24, c = 66 // 值为 66
```
::: info
注意，在特定语境下，逗号不会被解释为逗号表达式。例如：
```cpp
int a = 42, b = 24;
```
这里的逗号不是逗号表达式，而是用来分隔不同的声明的。

除此之外还有函数调用、初始化列表等语境下，逗号也不会被解释为逗号表达式。这些内容会在后续章节中介绍。
:::
